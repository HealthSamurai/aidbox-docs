# Developer Docs

Developer-facing reference for the CMS-0057-F API suite. Each API has its own page covering the base URL, the authentication flow, supported FHIR operations, search parameters, and request/response examples.

* [Provider Access API](provider-access-api/README.md) — backend services authentication, member matching, bulk data export.
* [Payer-to-Payer API](payer-to-payer-api/README.md) — opt-in member matching with persisted consent and bulk historical data export between payers.
